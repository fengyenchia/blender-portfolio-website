1. 先建立一個貝茲曲線，當作之後鏈節要走的路徑
2. 為了讓 chain 在線上平均分布，需要先用 resample curve 將線上的點增加並重新均勻排列，用 trim curve 可以調整線的長度
3. 接著製作 instance on points 中 instance 的部分
- 我用四邊形 curve 來當作主體，並用 curve to mesh 將曲線變成實體，profile curve 的輸入值是 curve circle。
- 接著加入表面細分，將 chain 的角變的圓弧一些
4. 接著製作 instance on points 中的 rotation
- 首先先用 curve tangent 搭配 align rotation to vector，可以讓 chain 旋轉，使它們沿著曲線的切線方向排列
- 為了要將 index 是奇數的 chain 旋轉，使得奇偶數的 chain 可以呈現環環相扣，需要加入一個 rotate rotation，並將模式轉成 local。透過 floored modulo 2 將奇偶數 index 輸出成 1、0，並乘以需要旋轉的數值與用 combine XYZ 指定旋轉方向
5. instance on points 中的 scale 和上課做的一樣，不過目前還不需調整曲線粗細（同時也是 instance 大小），所以我將 float curve 的數值都調成 1，這樣 instance on points node 都設置完成了
6. 再來用 menu switch 將四種材質變成可以在 UI 中切換的
7. 製作完成後，就將此 circle chain 的節點組成群組，命名為 circle chain links，並設置好可調式參數區

1. 和前一個鏈節一樣，先建立一個貝茲曲線，當作之後鏈節要走的路徑；加入 resample curve 來調整曲線細節
2. 接著要做 instance on points 的 instance，因為腳踏車鏈節有分內外兩部分，所以我分兩部分做
- 首先是 bicycle chain_inside，進入此 node group（圖3）後可以看到 chain inside 的 node group，這是上下兩個鏈片的部分。
- 進入 chain inside （圖4） 後，我先用兩個 curve circle 和 fill curve 來建立圓圈，底下是用 quadrilateral 來建立四邊形，並用 join geometry 合併，再用之前上課做的 extrudeNew 來擠出厚度
- 因為鏈片中央切口是圓弧的，所以我用 mesh boolean 把原來的模型切掉 兩個圓
- 因為調整模型厚度的同時，模型位置也會跑掉（不在中心），所以最後加上 transform geometry 來微調位置
- 此時 chain inside 就做完了（圖7），將這些節點群組起來，並回到上一層（圖3 那層）
3. 因為鏈片有分上下兩層，是對稱的，所以我調整位置時要鏡像調整。用 combine XYZ 單一調整 y 軸位置，下鏈片要加上 multiply -1 來達成反向移動（圖3）
4. 接著在同一層級（回到圖8）製作滾筒，透過兩個不同半徑的 cylinder 用 mesh boolean 相減，可以得到中空的圓 筒。因為滾筒有左右兩個，所以另一個用 transform geometry 調整位置，並用 join geometry 來合併
5. 把剛剛做的鏈片與滾筒都加上材質，並用 menu switch 讓材質可以透過 UI 來選擇，最後用 join geometry 合併兩部分
6. 因為整個外鏈節需要可以調整大小，所以加上 transform geometry 並用 combine XYZ 來指定要調整的軸向，如圖10
7. 最後將這些 node 群組成 bicycle chain_inside，設置好可調式參數控制區，如圖11
8. 再來要做 bicycle chain_outside 的部分，進入此 node group 後，可以看到和剛剛內鏈片一樣的節點，且製作過程也是一樣的，所以就跳過此部分（圖12）
9. 接著是做軸的部分，我用 cylinder 並將半徑調小一些、depth 調大一些，並將兩部分的軸用 join geometry 合併
10. 接下來和剛剛一樣，給予模型材質，並用 join geometry 將軸與鏈片合併
11. 合併完後和之前做法相同，為了調整 chain 的大小，加上 transform geometry 並用 combine XYZ 來指定要調整的軸向，如圖15
12. 製作完後就將這些 node 群組成 bicycle chain_outside，設置好可調式參數控制區，如圖16
13. 為了要將內、外鏈節輪流放在點上，變成一個完整的鏈條，我問了 ai 詳細的做法， 如圖17、18 。透過 geometry to instance 將內外兩部分鏈節分別變成兩個 instance，再用 join geometry 合併。因為剛剛有用 geometry to instance，所以現在輸出不是一個 instance，而是一個清單，裡面有兩個分開的 instance，並且賦予 index=0 和 index=1
14. 接著把 pick instance 打開，在 instance index 的輸入口接上判斷 index 是奇、偶的節點
15. scale 輸入口也是和上面的一樣，至此 instance on points 就完成了，將所有 node 群組成 bicycle chain_outside，設置好可調式參數控制區，如圖21、22
16. 實際成果如圖23、24
17. 最後將兩個做好的鎖鏈加上 realize instances，並用 menu switch 做成可切換鎖鏈類型的選單，就完成了！
