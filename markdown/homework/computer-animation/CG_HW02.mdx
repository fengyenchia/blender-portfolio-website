### step 1-1：葉子基本形狀

  label="step 1-1 葉子基本形狀"

1. 首先先畫一張草圖幫助自己思考，要怎麼長出葉子的各個部分。
2. 將葉子的參考圖放進檔案中，製作時可以對照著看。
3. 我先製作上面 4 個面，也就是葉片的部分，不包含底部心形。
4. 透過 Extrude Mesh 將面擠出。將 Vector 節點接到 offset，可以決定擠出的軸向，並調整偏移量。
5. 透過 Scale Elements 調整單一軸向的寬度。
6. 大致寬度調整完畢後，加上表面細分，就有葉片的感覺了。

### step 2-1：葉柄製作、調整葉基位置

  label="step 2-1 葉柄與葉基位置"

1. 向 `-Y` 方向擠出葉柄。
2. 將葉基整體往上提，製作心型葉片。
3. 在葉柄節點前新增調整葉基高度的節點，先調整高度，再長出葉柄。這樣葉柄最下方就可以跟著葉基移動而移動。
4. 為了讓葉基移動得更自然並避免模型重疊，我將移動的點分為兩部分：`16-18` 和 `50、53`，並透過 Math 節點改變兩部分移動的數值。當 `16-18` 往 Y 方向移動 1 倍時，`50、53` 會往 Y 方向移動 0.5 倍。

### step 3：細節調整：葉片下緣位置、葉尾寬度調整

<ImageCarousel
  label="step 3 葉片下緣與葉尾寬度"
  images={[
    { src: "/images/homework/computer-animation/hw02/w2-5-1.png", caption: "調整葉片下緣位置" },
    { src: "/images/homework/computer-animation/hw02/w2-5-2.png", caption: "葉尾形狀有點方" },
    { src: "/images/homework/computer-animation/hw02/w2-5-3.png", caption: "讀取點座標" },
    { src: "/images/homework/computer-animation/hw02/w2-5-4.png", caption: "計算 X 方向偏移量" },
    { src: "/images/homework/computer-animation/hw02/w2-5-5.png", caption: "葉尾寬度調整後" },
  ]}
/>

1. 我覺得目前 `0、2` 這兩點，也就是心型部分太開，所以想將兩點往中間移動。
2. 為了算出 X 方向的偏移量，先透過 Position 節點讀取點位置，再單獨取出 X 座標數值。
3. 接著將 X 座標數值乘以 `-0.5` 算出偏移量。負號是因為左邊的點要往右，右邊的點要往左移動。
4. 最後透過 Combine XYZ 節點指定 X 方向的偏移量。
5. 目前這片葉子的下半部有點方，所以回到前面，在表面細分之前新增葉尾寬度調整的節點，將 3 號邊變寬、將 2 號邊變窄。

### step 5-1：最終設定：Pivot 位置與葉片厚度

<ImageCarousel
  label="step 5-1 Pivot 位置與葉片厚度"
  images={[
    { src: "/images/homework/computer-animation/hw02/w2-7-1.png", caption: "嘗試調整整體位置" },
    { src: "/images/homework/computer-animation/hw02/w2-7-2.png", caption: "AI Studio 給的思路" },
    { src: "/images/homework/computer-animation/hw02/w2-7-3.png", caption: "AI Studio 給的思路" },
    { src: "/images/homework/computer-animation/hw02/w2-7-4.png", caption: "Pivot 與厚度設定" },
  ]}
/>

1. 要將 Pivot 位置調整到 `(0, 0, 0)`，我一開始想到的是透過開頭接在 Grid 後面的 Set Position 節點來調整整體位置。不過因為不知道 Pivot，也就是 86 號點的精準位置，所以怎麼調整都對不到原點。
2. 我詢問 AI：要如何得到 86 號點的 Y 座標，並將整體葉片向上移動 Y 值，使 86 在原點 `(0, 0, 0)`。
3. 解法是新增 Bounding Box 節點，用來讀取葉子邊界座標，並將最小 Y 值，也就是 box 底部取出。
4. 再新增 Set Position 節點，把剛剛讀到的數值乘以 `-1`，得到大於 0 的偏移量。
5. 最後透過 Combine XYZ 節點，將葉子整體向正 Y 方向偏移。
6. 透過詢問 AI 學到：Bounding Box 和 Set Position 這兩個步驟是分開進行的，前者取得偏移量，後者進行整體偏移。
7. 最後用 Extrude Mesh 節點，將葉片整體擠出厚度。

這是我第一次用 Geometry Node 來建模。過程中透過參考上課時製作的檔案來釐清各節點的功能，有批次存檔讓我在學習時非常清楚目前的步驟；卡關時也會詢問 AI Studio，幫助我思考與解決問題。雖然邊學習邊製作花了比較久的時間，但是完成後有很大的成就感。

透過 Geometry Node 來建模，讓我了解到用這種方式雖然速度較慢，但非常有邏輯、步驟也很清楚。最重要的是可以動態且精準地調整參數，不用怕做完後反悔卻無法調整。

**過程中遇到的困難：** step 2-1 在思考要如何製作葉基兩邊的心型葉片時，首先想到的是將葉柄旁的兩個邊向下擠出。不過實際製作後發現，它並沒有照著我指定的邊去長，而是長在其他邊上。於是我換個思路，改成調整葉基位置，將葉基往上提，就能產生心型葉片，而且這樣做的葉片模型或許會比一開始的方式自然一些。
