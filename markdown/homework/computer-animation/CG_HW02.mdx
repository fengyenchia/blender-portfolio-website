1. 因為我覺得目前 0、2 這兩點（心型部分）太開了，所以想將兩點往中間移動。為了算出 x 方向的偏移量：
- 先透過 position 節點讀取點位置，再單獨取出 x 座標數值
- 接著將 x 座標數值乘以 -0.5 算出 偏移量（負號是因為左邊的點要往右，右 邊的點要往左移動）
- 最後透過 combine XYZ 節點指定 x 方向的偏移量
2. 目前這片葉子的下半部有點方，所以我回到前面，在表面細分之前新增葉尾寬度調整的節點，將 3 號邊變寬、將 2 號 邊 變窄

### step 5-1：最終設定：Pivot 位置與葉片厚度

<ImageCarousel
  label="CG_HW02 slideshow 8"
  images={[
    { src: "/images/homework/computer-animation/hw02/w2-7-1.png", caption: " " },
    { src: "/images/homework/computer-animation/hw02/w2-7-2.png", caption: "ai studio 給的思路" },
    { src: "/images/homework/computer-animation/hw02/w2-7-3.png", caption: "ai studio 給的思路" },
    { src: "/images/homework/computer-animation/hw02/w2-7-4.png", caption: " " }
  ]}
/>

1. 要將 Pivot 位置調整到 (0,0,0)，我一開始想到的是：透過開頭（接在 grid 後面的）set position 節點來調整整體位置，不過因為不知道 Pivot （86 號點）的精準位置，所以怎麼調整都對不到原點
2. 我詢問 ai：要如何得到 86 號點的 y 座標，並將整體葉片向上移動 y 值，使 86 在原點（0,0,0）。得出的結果如圖二、圖三
- 新增一個 bounding box 節點，用來讀取葉子邊界座標
- 並將最小 y 值（也就是 box 底部）取出
- 再新增一個 set position 節點，把剛剛讀到的數值乘以 -1（因為 y 值為負），得到大於 0 的偏移量
- 最後透過 combine XYZ 節點，將葉子整體向正 y 方向偏移
3. **透過詢問 ai 學到：** bounding box 和 set position 節點這兩個步驟是分開進行的，從前者得到偏移量，從後者進行整體偏移
4. 最後用 extrude mesh 節點，將葉片整體擠出厚度

這是我第一次用 Geometry Node 來建模，過程中透過參考上課時製作的檔案來釐清各節點的功能（有批次存檔讓我在學習時非常清楚目前的步驟），在卡關時也會詢問我的 ai studio 幫助我思考與解決問題。雖然邊學習邊製作花了比較久的時間，但是完成後有很大的成就感！
透過 Geometry Node 來建模，讓我了解到用這種方式雖然速度較慢，但非常有邏輯、步驟也很清晰，最重要的是可以動態且精準地調整參數，不用怕做完後反悔卻無法調整。
**過程中遇到的困難：**
**step 2-1：**在思考要如何製作葉基兩邊的心型葉片時，首先想到的是：將葉柄旁的兩個邊向下擠出。不過實際製作後發現，它並沒有照著我指定的邊去長，而是長在其他邊上。於是我換個思路：改成調整葉基位置，將葉基往上提，就能產生心型葉片了，而且這樣做的葉片模型或許會比一開始的方式自然一些。
