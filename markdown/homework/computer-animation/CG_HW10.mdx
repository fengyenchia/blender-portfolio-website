## 作品技術概念

以六種 Primitives 作為 particle，以七種 Primitives 作為基底模型，製作 Particle Effects，並展現 Particle 散射與聚合的動畫。各物件的切換需設置 Menu Switch Node，讓使用者能夠切換不同的 p rimitives、meterial、particle motion 。
作品環境以 Volume Light 搭配 Noise Texture 遮光片進行打光（使用 EEVEE 引擎），Animation 總共有 120 格（5秒）。

### step 1-1：製作 particle effect

1. 首先，透過 distribute points on faces 在 mesh 上隨機分布點，點的數量（density max）暫時設為 500，並連到 group input 。seed 也 連到 group input，等等下 keyframe 時會用到。
2. 加入 instance on points，將剛剛的 node 連到 points 上。要改變 points 上的 instance，需要加入一些會用到的 primitives，不過目前先不在這裡加，之後會讓它變成 Menu Switch，所以先將 instances 連到 group input。
3. 底下的旋轉與縮放節點就和 [HW07](hw07.html) 一樣，透過 scene time 搭配 random value，讓 每個 particle 在每個 frame 的旋轉值都不同，縮放值也是透過 random value，讓畫面中 particle 都大小不一。
4. 再來是設定 particle 的 散射與聚合 ，可以看到圖2右邊有兩塊淺藍色的，這是我設定兩個不同模式的 particle motion，上面是會受到音樂影響動態，而下面不會。
- 先製作下面的 node，新增一個 set position，接著要改變它 offset 的部分。首先，新增 noise texture 並將 senen time 接 multiply（底下 value=0.000005）再接到 distortion 製作出 particle 原地抖動效果，再把 color 接口接到後面 multiply 的其中一個 vector，另一個要接到 group input，調整數值可以發現 1 會散射，0 會聚合。
- 上面部分也是類似，只是要用音樂來控制 particle 的位置，所以將 offset 的 input 做了些調整。我希望可以用一個值（一個 keyframe）同時控制 XYZ 的位置，所以先用 seperate XYZ 單獨調整一個維度的值，再將此值連到 combine XYZ 的三個維度上，後面就上底下的 node 一致。
- 最後加上 Menu Switch，將兩個 set position 連到選項中，再將最上面的 menu 連到前面的 group input，重新命名一下就完成選單切換了。
5. 開始加入音樂控制，先從最上面加號選取 video editing，將需要的音樂拉入底下時間軸。再回到 geometry node，將剛剛 seperate XYZ 中的 x 打一個 keyfrme，要記得保持選到這個 node，接著在底下時間軸切到 graph editor，按 channel &gt; sound to samples，選擇要影響 particle 的音樂（和剛剛匯入的一樣），然後就完成了。
6. 把這個帶有許多控制項與切換選單的 node 群組，命名為 particle effect。

1. 要看到 particle effect，需將剛剛製作 particle effect node group 的 input 接上許多 mesh 和 instance，且需將它們做成選單，我挑選了一些 primitives，並將它們都連到 menu switch 上，再分別接到 mesh 和 instances 的 input 上。
2. 在 instances 的 p rimitives 上，因為它們是 particle 本人，所以我做了更細部的調整，將每個形狀的參數（半徑、高度等）都加到可調式參數區，方便調整。
3. 再來製作 particle 的材質，我選了紅黃藍並用 menu switch 做切換，不過還有更細部的調整。
- 切換到 shading 面板，先將紅色材質加上 texture coordinate、seperation XYZ（選一個想要漸層的方向）、再用 color ramp 做漸層
- 黃色則做一個不一樣的效果，加入 texture coordinate、noise texture，可以做出噪波紋理（可在細調 noise texture 的參數），再用 color ramp 調整黑白噪波的顏色。
4. 最後回到 geometry node，加入 transform geometry 調整整體大小，而旋轉部分我想讓它自動轉，所以就用 scene time 來控制，並用 combine XYZ 讓　xy 方向都轉，multiply 可以控制旋轉速度。

1. 新增一個聚光燈，將功率調大，照亮 particle 們，如圖 1、2
2. 再來要製作體積光
- 新增一個 cube，將 cube 放大到可以完全包住整個場景（包含 particle、燈光）
- 打開上方 shading 面板，並將右側材質中的 surface 改成 none，volume 改成 volume scatter，最後將 density 調小，直到場景中 cube 變成半透明、避免太霧，改變體積顏色後，體積光就做好了，如圖3、4
3. 再來要將燈光變得有紋理，新增一個平面，移動至聚光燈**光源**處
- 在 shading 面板中開始加入材質
- 使用 noise texture 可以製作噪波紋理，並透過 color ramp 將黑的變更黑，白的變更白，讓它們的界線變得明顯一些
- 這樣就完成燈光紋理了，最後將 color 接到 principled BSDF 的 alpha，可以讓黑色部分消失（變透明），讓之後燈光照射可以存在透光與不透光的部分
4. 為了讓之後移動燈光時，紋理可以跟著移動，可以先選中材質平面，再選中燈光，按下 ctrl+p 可以設定父層，之後移動燈光時它就會帶著平面一起移動了，如圖7
5. 因為目前不夠亮，所以我加入一顆日光，放在 (0,0,0)，並調整燈光強度
6. 調整攝影機位置，照到 particle 們，如圖9，可以關閉上面的 viewport overlays，讓場景變乾淨
7. 最後在右邊 world 區域可以調整場景的顏色，我把整體調黑一點
