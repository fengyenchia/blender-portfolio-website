## 作品技術概念

用 Distribute Points On Faces Node 與 Instance On Point Node，製作政大 Logo 之 Particle 動畫。粒子在三維空間中運動，最後集合在一起形成政大 LOGO 的形狀。
環境以 Volume Light 搭配 Noise Texture 遮光片進行打光（使用 EEVEE 引擎），並在動畫中展現鏡頭語言。
Animation 總共有 192 格（8秒），影格圖輸出 Resolution 為3840 x 2160，格式為PNG；影片輸出 Resolution 為 1920x1080。

1. 現在將 logo 匯入 blender 中，右鍵選取 Trace Image To Grease Pencil，之後就擁有一個完美的曲線了
- 此時原本的 logo 就可以刪掉
- 圖2 是我問 ai Trace Image To Grease Pencil 用途的結果：它可以將圖轉成曲線（透過辨識明顯、高對比度的邊緣來取得線）
2. 因為此物體還不是 mesh，所以要先將它轉成 mesh
3. 轉完後它只剩下邊線，所以要進入編輯模式，全選後按 F 將 logo 中間填滿
4. 接著進入 geometry node，首先透過 Distribute Points On Faces 來隨機 生 成點在面上，使用 poisson disk 可以避免點太過於擁擠（它們彼此會有一定的距離）
5. 加入 instance on points，將製作好的葉子拖進這裡面，當作 instance（要勾選 as instance），也就是之後的 particle，並設定一個材質給它
6. 要讓 particle 隨機轉動，可以加入 random value，並用 scene time 的秒數控制它的 max 值（意思是每秒 max 都不一樣，視覺上就能達成持續在轉的效果）。要用 add 來將 這個向量加在 rotation 上，並可用 multiply 控制 rotation 的速度
7. 接著要調整 particle 的大小，一樣用 random value 隨機調整
8. 接著加入 set position，用 combine XYZ 連接 offset，並將 random value 範圍設定在 0~0.5，接到 z 方向上。使 particle 可以在 z 軸的 0~0.5 隨機分布（如圖8）
9. 再來要設定 particle 收縮，透過將 noise texture 的 color 向量與 0~1 的向量相乘，並接在 set position 的 offset 可以改變 particle 的位置
- 乘 (1,1,1) particle 會變遠 =&gt; 發散
- 乘 (0,0,0) particle 位置不變 =&gt; 留在原地
- color 其實是 particle 的位置向量，所以再 乘以 (1,1,1 ) 代表 particle 離原點的距離越來越遠
- 圖10~12 是我問 ai 此節點接法的結果
10. 最後將製作好的 node 群組起來

1. 首先加入一個平面，當作 animation 的背景，並加上黑色材質
2. 新增一個聚光燈，將功率調大，照亮 particle 們
3. 再來要製作體積光
- 新增一個 cube，將 cube 放大到可以完全包住整個場景（包含 particle、燈光、平面）
- 打開上方 shading 面板，並將右側材質中的 surface 改成 none，volume 改成 volume scatter，最後將 density 調小，直到場景中 cube 變成半透明、避免太霧，改變體積顏色後，體積光就做好了
- 如圖4
- 圖5 是我問 ai 關於體積光設置的結果
4. 再來要將燈光變得有紋理，新增一個平面，移動至光源處
- 在 shading 面板中開始加入材質
- 使用 noise texture 可以製作噪波紋理，並透過 color ramp 將黑的變更黑，白的變更白，讓它們的界線變得明顯一些
- 這樣就完成燈光紋理了
5. 為了讓之後移動燈光時，紋理可以跟著移動，我們先選中材質平面，再選中燈光，按下 ctrl+p 可以設定父層，之後移動燈光時它就會帶著平面一起移動了，如圖ㄚ
6. 調整攝影機位置，照到 particle 們，為後面動作做準備
