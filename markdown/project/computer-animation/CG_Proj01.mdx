## 作品技術概念

3D Printing
1. 將 logo 模型再次修改，讓它不只是 2D 的樣子，而是以立體的角度去思考，並實際調整出來。最後以 Measureit 測量模型尺寸，並展示各角度圖片。
2. 為模型設計可拆卸或可活動機構（Articulate），讓模型與平面底盤銜接、模型部位拆解共有至少四個分開的部分。並實際以簡單幾何體製作出結構，最後準備好 in scale 的 3DPrinting Ready 的模型。
Animation
1. 製作動畫的分鏡腳本，至少包含 6 顆鏡頭。
2. 延續先前製作的 Procedural NCCU LOGO Animation， 以 Geometry Nodes 製作 含音樂的 片頭動畫，包含該 Animation 的 Breakdown，共不超過 60sec。內容包含 Particle Effects、Mechanical Ideas。 Animation時長至少三十秒(F=720)，Breakdown時長至少三十秒(F=720)。
3. 完成最終 Animation 的 3D 模型，延伸為 3D 列印實體的版本。
4. 這次 project 我製作了一個充滿秘境感的建築，讓建築慢慢的生長，周圍充滿小光點來營造氛圍。
**作品理念**
此作品是以政大 logo 為出發點，將 logo 延伸製作成有立體感的模型，初步模型製作過程在 [HW04](hw04.html) 可以看到。
我將模型設計成由許多植物（樹葉、樹枝、藤蔓）與生物（螢火蟲）所組成的建築物，底部附有座位區，供前來的觀眾可以休憩。四周與建築物中央底部充滿螢火蟲，打造出環境是個夏日夜晚的休憩場所的感覺。
模型中也設計了一些可活動機構， 透過各部分的解構或旋轉設計，讓整體變得有機，變得生動。

在思考模型的 Articulate 之前，我想先將原先模型不夠立體的模型修改一下，讓之後的製作比較有想法。現在它只是校徽的形狀，但我想做一個高樓感的物體，所以我上網搜尋[「螺旋建築」](https://www.google.com/search?q=%E8%9E%BA%E6%97%8B%E5%BB%BA%E7%AF%89)，看看相關圖片來累積靈感。
**以下為模型修改過程：**
1. 首先，我先將圓環與花瓣調高，變成一個立體高樓，再加上一些 cylinder 的結構
2. 接著取消中間圓環，將原來的支撐縮短、複製很多個，但發現這樣電腦會很卡，所以需要修改方向
3. 我將圓柱拉長，並設定它們的高度是由小至大，再將圓環接回去。經過陳時間思考與修改，我覺得目前（圖4）的樣子還算滿意。
4. 新增的圓柱節點如圖5，先將 cylinder 匯入，再透過 transform geometry 調整尺寸與角度。接著用 repeat node 複製物件，過程中調整物件的角度與大小，讓整體有階梯式往上的感覺
5. 因為目前文字是懸空的，原先的想法是增加一個圓柱給它，但剛好這週要思考模型的可活動機構，所以上網看了一些案例，就發現了這個影片（[https://youtube.com/shorts/dTGDKcJudHs?si=KymIS_6TPrBLmom4](https://youtube.com/shorts/dTGDKcJudHs?si=KymIS_6TPrBLmom4)），可以做類似螺旋紓壓玩具的結構。不過網路上沒有找到教學影片，就從這部短片中拆解它的結構來學習，把它運用在我的模型上（這會成為之後設計 Articulate 的其中一個結構 ）。
6. 做好的結構如圖6、7、8，我將結構頂、底部加上輔助旋轉的圓盤，並將文字置於圓盤上，希望之後測試時能夠順利列印
7. 再來是螺旋結構的節點設計
- 先將 spiral 的主體製作出來，再透過 repeat node 每隔 120 度複製出下半部分的螺旋，上半部分也是同樣的步驟，只不過整體要多轉 60 度，讓螺旋結構可以交錯。
- 再來設計圓盤，我用 cylinder 產生一個圓柱體，再將內側縮小。因為我放進切片軟體測試時發現，若內外半徑都相等，會有大量的支撐要印，所以要讓半徑從中間螺旋大小慢慢放大到文字大小
- 因為螺旋上方不是平的，所以要透過 mesh boolean 來切，我用圓盤當作減去的對象，把螺旋都切平整
- 最後用 mesh boolean 取聯集就做好了
8. 為了將各物件的底部都對整齊，我會將圖13的節點加入各物件 node group 的偏後面，用於將底部設為 z=0，來進行整體物件位置的調整，這樣列印出來也會更好看
9. 為了之後可以更靈活的控制，我將每個物件的可調式 參數 控制區整理了一下，把各物件都再分群放好，最後微調數值， 模型就大致調整好了！
10. 最終成果如圖15，不過目前花瓣部分還是未和其他東西連接的，之後會設計 Articulate 來完成這部分

1. 為了要產出 Printing Ready Model，需要在 blender 做一些設定
- 預設 blender 會以公尺作為單位，所以須先在場景終將單位設置成 millimeters，並將 unit scale 改為 0.001，這才是切片軟體上的尺寸單位
- 不過調整後視圖區的網格距離就變大了，這樣不好對尺寸。所以要將 viewport overlays 中，grid的尺寸調小 1000 倍
- 這樣就設置完成了
2. 再來是確定原點在 (0,0,0)，目前 3D 游標在 (0,0,0)，所以只要按滑鼠右鍵，將原點設置在 3D 游標即可
3. 這樣就完成 Printing Ready Model 了，放進切片軟體檢查可以看到，尺寸與 Blender 上看到的一致
4. 接著透過 Measureit 來展示尺寸，但因為這個插件是在編輯模式下選擇網格的點來測量，而目前用 GN 做出模型 不是真實存在的網格，所以要先另外存一個檔，再把 GN 的修改器 應用，才可以開始量。

- 第一個 我先製作可以整片轉動的結構，以球形當作旋轉體，像是關節一樣，可以 360 度旋轉

我想要營造出秘境感，所以會用到許多 particle effect 來製作螢火蟲

## 自我評量與作品建議

**[week06] 9 /10 [week08] 8 /10**

**3D Printing**
**[week06]**原本只是單純的 logo 模型時，因為很單調，難以思考它可以有的可活動機構。但調整完模型成立體建築結構後，發現它可以變得很有趣，很多想法也隨之出現。此外，要設計出可活動機構不難，有難度的是印出來結構之間不會卡死，所以需等待上課時實際列印才能得知自己有沒有設計好。總結來說我對這次的 logo 模型修改很滿意，也期望之後它能夠順利印出來！
**[week08]**在印製實際 logo 模型前，我先製作幾個簡單的結構並印製出來測試看看，發現最難的是：結構本身不能太大，因為它是要放在模型裡面的，但結構的公差不能太小，否則無法順利活動。這部分我來來回回改很多次，讓公差維持在 0.5~1.0mm 左右。
**Animation**
**[week08]**這次在做動畫時，因為有先寫好分鏡，所以製作起來會比先前順利，不過因為太卡了，沒辦法在視窗中看到動畫實際動的過程，只能先想像，渲染完後才能真正看到，所以也沒有花很多時間在調細節。
**過程中遇到的困難：**
大概就是圖片要渲染很久，我調整了一些設定（看看能不能降低品質加速渲染），結果還是很久...
