1. 因為作業要求有提到，齒輪需要可以進行切換， 而我的模型中有許多齒輪，要幫每個齒輪各加一種樣式很麻煩，所以我改成先製作一個可切換齒輪樣式的 gear node group，其中圖1 是每個齒輪會重複用到的節點，所以也將它們加進 node group 裡，方便重複利用
2. 再來是加上更多樣式，我用 menu switch 做樣式切換，第一個是 normal，我讓它保持原本製作好的齒輪的樣子，接下來是其他樣式
- type A：type A 我將齒輪上打 4 個洞，透過 curve circle 做出基本形狀，再用 repeat node 邊旋轉 90 度邊複製 4 個，最後和原本的齒輪形狀用 join geometry 合併再用 fill curve 填滿，就完成了（如圖3）
- type B：type B 我將齒輪上放 6 個半圓，透過 arc 做出基本形狀，再用 repeat node 邊旋轉 60 度邊複製 6 個，最後和原本的齒輪形狀用 join geometry 合併再用 fill curve 填滿，就完成了 （如圖5 ）
3. 最後將所有 node 群組，命名為 gear switch，可調式參數控制區如圖7
4. 圖8 是實際將原本齒輪模型中的重複節點，改成 gear switch node group 的樣子，其中有可調式參數 gear switch，可以切換齒輪樣式

1. 鐘面我用 grid 製作，並用 extrudeNew 擠出厚度。因為鐘面上方放的是年份倒數（只會露出一個孔顯示幾年），所以上方我再用 grid 開一個洞，用 mesh boolean 減去
2. 再來是製作秒分時天的小鐘面，圖2 是合併的結果，之後會有詳細製作步驟
3. 圖3 是 hour node group
- 因為我要在圓周上平均分布 4 個點（指向 3、6、9、12 小時），所以先用 curve circle，resolution 設為 4，再將輸出接到 instance on points
- instance on points 的 instance 要放數字，首先製作數字 12。我用 string to curve 做數字，再將 index、equal 接到 instance on points 的 selection，代表 index = 0 的 instance 是 12（實際 index 等於多少，可以在製作時看鐘面上數字出現在哪個點，再調整）
- 其他數字製作過程一致，只差在 index 不同
- 做好所有數字後，就將它們用 join geometry 合併，再用 fill curve 填滿，並用 extrudeNew 擠出厚度
- 最後做一個小鐘面，用 curve circle 製作，而中間我想加一個圓形的物件，當作指針的旋轉中心，所以就加上一個 UV sphere，就完成了
4. minute node group 的製作流程和 hour node group 一致，差在要放的數字與數量不同，這裡就不多描述
5. 圖7 是 second node group 的鐘面製作流程，因為數字製作流程一致，所以不多描述。而秒針的鐘面我做成中空的，所以就多加一個小的 curve circle 來挖洞
6. day node group 的製作流程和 second node group 一致，所以也不多描述
7. 圖8、圖9 是最終完成圖
8. 因為指針旋轉速度和齒輪相同（同軸），所以指針 node 我放在 gear 物件裡
- 圖10 是分針指針，我用 curve line 製作，加上 curve to mesh 後，以 quadrilateral 接上 profile curve，做一個長方形指針，最後接到和齒輪一樣的旋轉 node 中
- 圖11 是秒針指針、圖12 是時針指針，作法皆一致
- 而最後的天針不用放指針，不過需要做一個連接齒輪與鐘面的線段，所以圖13 我用 curve line 拼出直角線段
9. 圖14、圖15 是最終成果

這次材質以金屬為主，其中我在金屬上都加了一些細節，進入 shading 面板調材質
1. 第一種是生鏽材質，我將 [HW14](hw14.html) 做好的拿來用，這裡就不詳述
- 不過我將模型的生鏽部分加在鐘樓腳的下部分，而不是整體，圖1 是鐘樓腳的材質節點
- 我用 position + separate XYZ 只讓 z 方向決定生鏽程度；再接 map range 呈上調整生鏽程度的 rust change，其中高度 6 到 0.5 對應 0 到 1，代表上到下是不生鏽到生鏽
- 圖2 是完成圖
2. 再來是多種顏色的金屬材質加細節，我參考網路上的[教學影片](https://www.youtube.com/watch?v=bCgPjfSSN50)來製作，但和上課的流程其實類似
- 用 texture coordinate 開始，讓紋理可以根據模型在 3D 空間中的真實尺寸來均勻投影
- 顏色先用 mapping + noise texture 製造出紋理，其中 scale 的 z 方向調大，可以讓紋理有拉長效果，更像金屬刮痕；用 color ramp 調整兩色與它們的分界
- 金屬度維持 1
- 粗糙度一樣用 color ramp 調整成適合分界
- normal 用 color ramp 調整後，需要用 bump 去微調高度，將 strength 設為 0.01，讓表面微微凸出就好
- 圖6 是完整材質 node
3. 其他顏色的製作方式一致，只有 base color 前的 color ramp 中，顏色要調整成想設定的金屬顏色，像是金色設定如圖7
4. 發光材質我另外設置，如圖8，我將 emission 調成 0.5，同時要到 compositing 面板，加上一個 glare 才能看到發光效果（若沒有發光要檢查 viewport shading 中，compositor 要開成 always）
5. 圖11是完整加完材質的模型
