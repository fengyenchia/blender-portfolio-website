## 作品技術概念

用課程中學到的 Geometry Nodes 概念，針對「百年校慶倒數」的主題設計出一支約 30 秒之倒數計時機械鐘動畫，
其中 「百年校慶倒數計時鐘」 需符合主題、具備故事性、辨識度與延展性，同時達到以下五項要求：
1. 1-100 之倒數動畫呈現
2. 時鐘必須展示其齒輪構造， 其中齒輪需要以鏈條帶動
3. 必須能切換齒輪形式
4. 有 5 種以上的機械運作原理
- 鏈條傳動機構：用作時鐘齒輪與外部鐵門上齒輪傳動
- 齒輪與齒輪傳動：用作時鐘內部齒輪傳動
- 日內瓦機構：用作指針跳動
- 凸輪連桿機構：用作雨傘裝飾上下移動
- 往復滑塊機構：用作鐵門開
5. 所有機械與原理，請整合在一具模型，彼此連動、符合主題
此外，也為倒數計時鐘設計出多種的可調式參數，並整理在單一 UI 面板中提供使用者調整
**作品理念**
這件 作品是政大百年校慶 倒數計時機械式時鐘動畫，我 以鐘樓為出發點，製作包含政大特色的時鐘。
- 雨傘造型代表政大很常下雨（用凸輪機構帶動雨傘上下移動）
- 模型前方的鐵門緩慢打開，象徵歡迎校友回家（用往復滑塊機構帶動鐵門往左往右開啟 ）

**遇到的問題/解決方式**

**我遇到的問題有：**
1. 我想從 geometry node 裡將材質的參數加到可調式參數區，直接和其他參數和同一個 UI 中被調整。因為材質做好、參數設定好後，到 geometry node 裡用 s et material 指定材質，此時可調整的參數接口就沒有出現，所以我不能拉一條線到 group input 去調整。
- **解決方式** ：因為 shading 和 geometry node 面板屬於兩個不同 系統，所以需要用 a ttribute 讓這兩個系統可以溝通。
- geometry node ：在 s et material 前加入 store named attribute ，讓可調式參數區的數值可以被存起來，同時要將它取一個名字，之後讓 shading 可以順利接收
- shading：加入 attribute，將參數的 input 連上，並命名為剛剛取好的名字，這樣就可以讀取到 geometry node 存好的參數了
2. 當齒輪在 旋轉時，上面的生鏽材質不會跟著旋轉
- **解決方式：**在 transform geometry 前，新增 store named attribute，並將物體的 position 存進去，接著進入 shading，用 attribute 接收位置資訊，並用 mapping 映射位置座標到 noise texture 的 vector，這樣投影座標會隨網格一起旋轉，鐵鏽就可以貼合在齒輪上了
這次遇到的問題大多來自材質與 geometry node 之間的溝通，我透過問 ai 來學習欲解決問題。

**自己最滿意/最困難的部分，以及在製作該專案的獲得**

**滿意：**
這次模型的面數比期中少了一些，動畫也比較不卡一點。我透過將一些模型的 resolution 降低，像是把 cirve circle 的 resolution 從 32 降到 8，雖然外觀上沒有差很多，但面數減少了許多，等待 blender 算圖的時間也快了不少。
**困難：**
因為我的模型中有大量的齒輪傳動，所以一開始在設計齒輪擺放位置和計算傳動比時，花了一些時間。有時齒輪齒數決定好了，但位置、旋轉方向不對，就必須拆成兩個齒輪，或是調整齒輪位置，同時這樣就會連帶影響後面設計好的齒輪，所以此部分有花一些時間。
我覺得這次製作專案有比期中時順利，因為期中遇到過的問題，像是 render 很慢、將所有物件整合在同一個模型，這次都有吸取之前的經驗，所以順利解決。另外我覺得透過這一學期的練習，用 goemetry node 製作模型的速度快了不少，這次模型大多是為了專案而新做的（少有從平時作業累積拿過來的），所以需要花一些時間製作，但我覺得這次的進度還算控制得好，且成果也很滿意。
