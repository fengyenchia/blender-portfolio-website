## 作品技術概念

這個禮拜開始思考 project 02 要做什麼樣的機械時鐘、研究機械鐘構造與實際製作看看。
我想做包含： **100天轉一圈、****時針、粉針、秒針**的齒輪結構，轉速比為 1：200：2400：144000。

我透過這支影片來了解鐘錶內部的齒輪構造，以幫助我設計齒輪結構
[https://www.youtube.com/watch?v=24TrIRIJcaY](https://www.youtube.com/watch?v=24TrIRIJcaY)
從影片中我學習到：
1. 齒輪比 a:b，轉速比就會是 b:a
2. 此結構需要透過一個重捶來引入動力，來讓第一個齒輪開始轉動，從而連動其他齒輪。不過若只有 重捶，齒輪會快速旋轉將能量釋放，所以需要一個 [擒縱器](https://zh.wikipedia.org/zh-tw/%E6%93%92%E7%BA%B5%E5%99%A8) ，單擺以每秒擺動一次的頻率來卡住擒縱輪，讓重錘的動力可以每秒釋放一次

1. 首先用 star 建立一個星形曲線，這是齒輪的主體，再透過 points 設定齒數
- 加入一個 fillet curve，這可以將兩條曲線的相交銳角或鈍角轉換成圓弧狀，塑造出齒輪的樣子
- 接著透過 full curve 來填充曲線成面，最後設定材質與用 extrudeNew 擠出厚度就完成一個齒輪了
- 因為我是做雙層齒輪來減少體積與增加傳動比，所以需要在後面加上一個 cylinder 來讓兩個齒輪連動
2. 再來是讓齒輪旋轉
- 我用 transform geometry 來控制齒輪位置與旋轉，並透過 scene time 與 combine XYZ 讓齒輪隨時間沿 z 軸轉動，可以在加上 multiply 來控制速度
3. 因為每個齒輪的做法都是一樣的，所以就不再詳細描述後面內容。不過要特別注意的是，轉速比和齒數比成反比，所以齒數越多，轉動速度會越慢，藉此推得時針、分針、秒針和天針的齒數。
4. 圖 7~9 為成果圖，黃色齒輪為最終呈現的 時針、分針、秒針和天針
