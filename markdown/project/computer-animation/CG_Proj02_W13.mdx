每週進度更新

project 02 想做一個鐘樓造型的機械式時鐘，鐘樓頂端是雨傘，代表政大的特色「下雨」，鐘樓前方有校門，它會從關閉狀態慢慢向兩側開啟，直到倒數計時完成，大開打門也代表歡迎校友回家。
這週將模型大致完成，會用到的 機械運作原理有：
1. 鏈條傳動機構
2. 齒輪與齒輪傳動
3. 日內瓦機構
4. 凸輪連桿機構
5. 往復滑塊機構
詳細示意圖可以看 [project02_w11](project02_w11.html) 頁面。

上週只做完時分秒的齒輪，不過秒針我希望可以間歇性跳動，而不是等速旋轉，所以我加了日內瓦機構在內。同時，我設定整組齒輪的動力來源是天針（轉速最慢得當動力來源），所以將鍊條放在此上。
1. 首先製作有凹槽的日內瓦輪
- 基本造型和齒輪結構相似，所以也是用 star 來做，並擠出厚度
- 再來是減去中心圓柱，我用 cylinder 做出圓柱，並用 mesh boolean 減去
- 除了圓柱，還要再減去**鎖定圓弧**和**軸向凹槽**，我先用 curve circle 製作圓形，但因為要有 6 個相同形狀不同位置的圓弧，所以用 repeat node 來複製；軸向凹槽製作方式相同，只是形狀換成柱體
- 最後微調一下物體位置旋轉和大小就完成了，再將所有 node 群組成 Driven Wheel 的 node group，如圖6
- 圖7 是可調式參數控制區的內容，圖8 是目前 日內瓦輪 的樣子
2. 再來製作圓形的主動輪，它是動力來源，會帶動 日內瓦輪（ 從動輪）旋轉。 主動輪有分上下兩層，上層有 一個圓銷，它會轉進日內瓦輪的凹槽讓它轉動，轉了 60 度後又會離開日內瓦輪，使它停止轉動， 上層 還有一個鎖定凸輪，它會和日內瓦輪的圓弧咬合。
- 先製作上層，用 curve circle 製作圓盤，並用另一個 curve circle 減去一部分，變成月亮形狀的鎖定凸輪
- 圓銷也是用 curve circle 來製作，最後把這兩個部分用 join geometry 加在一起
下層是一個大圓，旁邊有一個小圓，上面放 圓銷
- 先用 curve circle 製作大圓和小圓，並用 transform geometry 微調位置
- 厚度的部分因為整體要和日內瓦輪一樣後，所以下層厚度是整體的 o.5 倍
- 製作好後也放入 join geometry，接著就可以前往下一部份，目前在圖6
- 一樣要減去中心圓柱，所以用 cylinder 製作柱體，並用 mesh boolean 減去它
- 最後微調一下位置旋轉和大小， 再將所有 node 群組成 Driving Wheel 的 node group，如圖13
- 圖14 是可調式參數控制區的內容，圖15 是目前 主動輪 的樣子

1. 首先是校門的外框，我用四邊形 quadriateral 做，並搭配 fillet curve 可以做倒角，再加入 curve circle 增加粗度，最後微調位置（讓它底部在 z=0）
2. 再來是校門內部的線條，我 是 用 mesh line 擺橫的，上面設定一些點（這部分是可調式參數），平均等分這條線；再用 instance on points 的方式，將 cylinder 加上去；為了讓我調整 count 時，門上的線條都可以和門的寬度均分，所以需要透過很多 math node 去計算（整體 line 的偏移量 + count*line 之間的間距 = 門寬度）
3. 最後把兩部分用 join geometry 加起來，而因為門是兩邊對稱的，所以我另一邊用原有的 node 加入一個 transform geometry 並設定 scale X = -1，就可以鏡射過去了
4. 校門移動的方式是透過往復滑塊機構，讓它往左右移動，所以接下來就是製作此結構。此結構分成齒輪、連桿和螺栓，如圖8
- 齒輪的製作方式和前面一致，所以就不再描述，不過可以看到兩側齒輪有分前後各兩個，後面齒輪被 時鐘齒輪的動力來源連動，接著因為同軸讓前面齒輪也可以跟著動
- 連桿我用 curve line 製作，其中連著齒輪的那點是作圓周運動，連著校門的另一點是做直線來回運動。圓周運動公式是 x=r*cos(angle), y =r*sin(angle)，所以我就著個這個結構去設定 math 節點，直線運動的那點會固定 y、z 方向，只需透過調整 x 方向來完成
- 最後微調位置並加上材質就完成了，左右做法類似但方向相反
- 螺栓我用 cylinder 製作，就把它擺在校門與連桿間、連桿與齒輪間就完成了，不過要記得它的運動方式和上面連桿的兩點相似，所以也是需要設定
5. 圖16、17 是完成後的圖片，圖18 是可調式參數控制區
6. 最後要記得在時鐘齒輪組中補上相互連動的齒輪，整體做完如圖21，箭頭方向為主動力到最後連動齒輪方向

3. 再來是中間方型平面，我用 cube 製作扁的平面，再用 transform geometry 調整位置
4. 最後做上面放置時鐘的立方體，主體用 cube 做，再用 mesh boolean 減去一個小一點的 cube，做出前方挖空感覺

## 自我評量與作品建議

**8 /10**

這週做了很多建模的部分，我預想先把模型做完，再想 分鏡和整體 動畫，會比較有想法。 裝飾性物件我是以 政大特色去思考，並想著如何搭配機械結構，最終想到可以做雨傘開合結合凸輪機構，同時也符合政大特色「下雨」。
齒輪部分我覺得個齒輪的位置安排最難，要避免擋到其他齒輪同時扣合相鄰齒輪來連動。第二難是要讓不同齒數的齒輪每一齒的大小相同，因為這樣才能正常計算傳動比並正常轉動，不過為了美觀和與讓齒輪間相互扣合，我有調整了一些齒輪的齒的大小，這樣有稍微不符合實際情況。
